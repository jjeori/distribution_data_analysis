# distribution_data_analysis

한국전자정보통신산업진흥회에서 주최한 제 2회 유통데이터 활용 경진대회의 비즈니스 아이디어 부문으로 프로젝트를 수행했습니다.
<br/>2023년 8월 ~ 9월 동안 진행되었습니다.
<br/>현재는 공모전에서 제공받은 2354만 건의 고객 데이터를 이전과 다른 관점으로 분석하고 있습니다.

-------
### **공모주제**
* 중소유통물류산업에 도움이 될 수 있는 아이디어

--------

### **문제 상황**
* 대기업의 거대 물류네트워크로 인해 중소 유통물류산업이 입지를 잃어가고 있다.
* 가장 큰 피해를 입고 있는 사람은 마트를 운영하는 소상공인이다.
* 대형마트, 마켓컬리 및 쿠팡, 편의점으로 인해 동네 마트를 이용하는 소비자들이 줄어들고 있기 때문이다.

--------

### **문제 정의**
* 동네 마트를 이용하는 소비자를 늘리기 위한 방안을 모색해야 한다.

--------
 
### **분석 목표**
* RFM 분석을 통해 슈퍼마켓의 차별성을 찾고 비즈니스 아이디어 제안하기

### **아쉬운 점 및 배운 점**
* 가장 큰 아쉬움은 핵심 타깃을 정하고 구체적으로 비즈니스 아이디어를 제시하지 못했다는 것
* 아래는 핵심 타깃을 선정한 뒤 전체적인 방향성에 대한 예시입니다.

  * 예시
   1. 결제 빈도와 결제 금액이 높은 연령대를 선택
   2. 이들의 카테고리별 이용 현황을 파악
   3. 이를 통해 소비 패턴을 찾기
   4. 해당 연령대의 소비 패턴이 있다면, 이를 슈퍼마켓과 연결할 수 있는 것이 무엇인지 고민하기

* 사고의 중요성
  * "이 데이터에 있는 모든 인사이트를 다 찾겠다"가 아니라, "이 데이터에서 근거를 찾아서 문제를 해결하겠다."는 사고의 중요성을 깨닫게 되었습니다.
